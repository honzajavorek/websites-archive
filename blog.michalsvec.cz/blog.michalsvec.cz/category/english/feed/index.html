Excellent combination of Apache, PHP and MySQL. When I was testing Ruby on Rails, I wanted to use my current MySQL databases from MAMP. It wasn’t so easy as I expected. Here is one possible sollution:

  1. Install MySQL with Homebrew. 
    brew install mysql
  2. Install MySQL gem 
    gem install mysql
  3. Set the socket in the database connection in database.yml to: 
        adapter:  mysql
    database: dbname
    username: user
    password: pass
    socket: /Applications/MAMP/tmp/mysql/mysql.sock
  4. Start Mongrel with 
    script/server

If you will run to troubles with this message:

Status: 500 Internal Server Error
uninitialized constant MysqlCompat::MysqlRes

run this command:

export ARCHFLAGS="-arch i386 -arch x86_64"
gem install mysql -- --with-mysql-dir=/usr/local --with-mysql-config=/usr/local/bin/mysql_config
I hope, this will help. If you have better sollution, let me know in comments.

If you click on mailto link, it opens new tab with your favourite webmail service.  You can add Hotmail, Yahoo mail, Gmail etc. Here is simple solution for  Gmail.com.

  1. Turn off Opera!
  2. Edit file webmailproviders.ini. On linux located in /usr/share/opera/defaults.
  3. Add following lines
    [Gmail.com]
    ID=8
    URL=https://mail.google.com/mail/?view=cm&fs=1&to=%t&su=%j&body=%m
    ICON=http://mail.google.com/mail/images/favicon.ico
  4. Save and close
  5. Start Opera, open Tools->Preferences->Advanced->Programs and edit mailto: protocol. Set your webmail as default

Now you can send mails with your webmail service instead of desktop app.

All we need is jQuery (jquery.com). With simple snipet, we can center whatever we want inside any element. The idea is simple: move child element to vertical center with margin-top.

Everything you have to do is put this simple code into head of your html file.

<script type="text/javascript">
$(document).ready(function(){
    $('.vc').each(function(){
        var thish = $(this).height();
        var parenth = $(this).parent().height();
        var topmargin = (parenth - thish) / 2;
        $(this).css('margin-top', topmargin);
    });
});
</script>

This should work on all elementswith class .vc.
